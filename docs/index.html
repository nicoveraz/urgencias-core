1. Carga tu CSV

Arrastra el archivo al área inferior o haz clic para seleccionarlo.

Formato esperado del CSV

Columnas obligatorias:

  • fecha_ingreso — formato YYYY-MM-DD HH:MM o YYYY-MM-DD HH:MM:SS
  • fecha_egreso — mismo formato

Ejemplo:

fecha_ingreso,fecha_egreso
2024-06-01 08:15,2024-06-01 10:47
2024-06-01 09:03,2024-06-01 12:22
2024-06-01 09:45,2024-06-01 11:10
